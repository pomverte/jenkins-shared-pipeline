= Jenkins Shared Pipelines

== Pipelines

. link:./vars/mavenPipeline.groovy[mavenPipeline.groovy]
. link:./vars/dockerPipeline.groovy[dockerPipeline.groovy]


== Use me

.Jenkinsfile
[source,groovy]
----
#!groovy
mavenPipeline {
  runUnitTests = 'false'
  nexusDeploy = 'false'
  ansibleDeploy = 'true'
}
----

.Jenkinsfile
[source,groovy]
----
#!groovy
dockerPipeline {
  pushDockerImage = false
}
----
